- # Chatty
- **Chatty** is a real-time chat application that enables users to communicate instantly with secure authentication, private chats, and a smooth messaging experience.

- ## Features
- Secure user authentication (JWT-based)
- Real-time one-to-one messaging
- Online/offline user status
- Media sharing support
- Instant message updates
- Protected routes & APIs

- ## Tech Stack
- **Frontend:** React, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Real-time:** Socket.IO  
- **Authentication:** JWT
